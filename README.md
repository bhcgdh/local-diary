# 本地日记

Windows 桌面月历日记。点击日期即可写日记，内容保存到本机 `data/diary.db`，并按年份同步导出为 Excel 可直接打开的 `data/2026年的日记.csv`。

窗口采用无边框桌面挂件样式。拖动顶部区域可以移动位置，点击“置顶”可以让日记保持显示在其他窗口上方。默认使用蓝色主题。

显示内容：

- 公历和星期
- 农历
- 年、月、日干支
- 年、月、日五行纳音
- 日志第一行默认填写当天农历和年月日干支
- 每个年份单独生成一张表格
- 表格包含“id”“时间”“日志”三列，id 是日期在当年的序号，每年从 1 重新开始
- 点击“同步印象笔记”，会在 `B1-日记` 创建以年份命名的笔记，正文直接显示“id、时间、日记内容”三列表格
- 印象笔记命令行接口不支持自动删除；同步后会打开同名笔记列表，旧版本需要手动删除
- 点击“打开日志文件夹”可以查看本地数据库和按年份生成的 CSV 表格
- 每个日期可以记录待办事项；月历使用 `✓` 标记有待办的日期
- 待办同步导出到 `data/待办事项.csv`，包含“时间、农历、年月日干支、待办内容”四列
- 月历日期格左下方显示当天日干支，天干和地支分别按五行着色：木绿、水蓝、火红、土为黄土色、金为亮金色

双击 `打开本地日记.vbs` 静默启动，不会显示命令行窗口。日记会在切换日期、点击“保存日记”或关闭窗口时保存。

